- .Net Core 5.0.12
- MVC
- MSSQL Server min versi 2012

Database akan ter-create otomatis beserta table-table nya saat aplikasi di running

- UserName: user1, password: user1, role: Penyewa

UI menggunakan bootstrap 5, UI belum 100% selesai
